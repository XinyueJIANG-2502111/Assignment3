# Erase Everything

> **Tap screen to Erase Everything.**

> システムの破壊とグリッドの消去をテーマにしたシンプルなタップゲーム。

---

## How to Play
1. **Tap to Erase**: Tap anywhere on the screen to trigger system errors and dissolve the grids.
2. **Chain Destruction**: Clear blocks before the system overflows.
3. **Control Settings**: Access the settings menu to dynamically adjust BGM and SFX volumes using sliders.

## プレイ方法
1. **タップで消去**: 画面をタップしてシステムエラーを誘発し、グリッドを消去します。
2. **連鎖破壊**: システムがオーバーフローする前に、アイテムを取り除きます。
3. **コントロール設定**: 設定メニューからスライダーを使用して、BGMとSEの音量を動的に調整できます。

---

## Tech Stack / 技術スタック

*   **Engine**: Unity 2022.3+
*   **Language**: C#
*   **Shaders**: HLSL
*   **Target Aspect Ratio**: Locked to 16:9 (supports dynamic window resizing)
